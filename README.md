# Discord Clone с голосовыми каналами

Клон Discord с реализацией голосовых каналов на базе WebRTC для общения с друзьями.

## Особенности

- Создание и подключение к голосовым комнатам
- Голосовое общение в реальном времени через WebRTC
- Текстовый чат в комнатах
- Индикаторы активности пользователей
- Управление микрофоном (вкл/выкл)
- Отключение звука (деafen)
- Простая аутентификация по имени пользователя

## Технологии

- **Frontend**: HTML, CSS, JavaScript (ES6+)
- **Backend**: Node.js, Express, Socket.IO
- **WebRTC**: SimplePeer для голосового общения
- **Real-time**: Socket.IO для чата и управления комнатами
- **Деплой**: Render (через GitHub)

## Установка и запуск локально

1. Клонируйте репозиторий:
   ```bash
   git clone <your-repo-url>
   cd discord-clone
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер:
   ```bash
   npm start
   ```
   или для разработки с автоперезагрузкой:
   ```bash
   npm run dev
   ```

4. Откройте в браузере:
   ```
   http://localhost:3000
   ```

## Деплой на Render

1. Создайте аккаунт на [Render](https://render.com)
2. Создайте новый Web Service
3. Подключите ваш GitHub репозиторий с этим проектом
4. Укажите следующие настройки:
   - **Name**: discord-clone
   - **Region**: ближайший к вам (например, Frankfurt)
   - **Branch**: main
   - **Root Directory**: оставьте пустым
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
5. В разделе "Environment Variables" добавьте:
   - `NODE_ENV` = `production`
6. Нажмите "Create Web Service"

После деплоя ваше приложение будет доступно по URL, который Render предоставит (обычно вида `https://your-service-name.onrender.com`).

## Использование

1. Введите ваше имя пользователя (минимум 3 символа)
2. Создайте новую комнату или присоединитесь к существующей по ID
3. Используйте текстовый чат для общения с другими участниками
4. Управляйте микрофоном с помощью кнопок внизу экрана
5. Приглашайте друзей по ID комнаты для голосового и текстового общения

## Настройка GitHub репозитория

1. Создайте новый репозиторий на GitHub
2. Инициализируйте Git в проекте:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/your-username/discord-clone.git
   git push -u origin main
   ```
3. Подключите репозиторий к Render для автоматического деплоя при изменениях

## Известные ограничения

- Нет постоянного хранения данных (комнаты, пользователи и сообщения существуют только во время работы сервера)
- Нет системы авторизации (только имя пользователя)
- Ограниченная функциональность управления аудио
